# Contributing to mem-hint

Patent Pending: Indian Patent Application No. 202641053160.

Issues and discussion are welcome, especially around:

- PMU event mapping for specific CPU microarchitectures
- CXL DVSEC register layout for real devices
- Integration patches for vLLM, TensorRT-LLM, and PyTorch

Pull requests are accepted for documentation, examples, and userspace library improvements.

Pull requests for [`kernel/`](kernel/) require discussion first because of interface stability and patent implications.

All contributors must agree that contributions are for research purposes and that patent rights remain with Manish KL.

Code style expectations:

- `kernel/`: Linux kernel style
- Python: PEP 8 / flake8-compatible
